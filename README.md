# ‚öíÔ∏è Collapsible Game Rules

**No Backports:** This mod targets **Minecraft 26.1+** (Snapshot 8). Older versions are unsupported.

> **Respect the Player's Time, Not the Game's Rules.**

**Collapsible Game Rules** makes the GameRules UI screens collapsible by category. Are you tired of scrolling through a massive, unorganized list of game rules every time you create a world or tweak settings? This mod solves the clutter by turning the standard Game Rules screen into a clean, easy-to-navigate interface.

## ‚ú® Features

### üóÇÔ∏è Collapsible Categories
Minecraft game rules are automatically grouped by their default categories (Player, Mobs, Spawning, Updates, etc.). Instead of a giant flat list, you get a clean, organized view.

### üñ±Ô∏è Toggle Headers
Click on any category header to instantly expand or collapse its contents. Find the settings you need without scrolling continuously!

### üõ°Ô∏è Client-Side Only
Works out of the box on your client. You do not need to install this on a server. Just drop it into your `mods` folder and enjoy a cleaner UI immediately.

### üß© Seamless Mod Support
Our collapsing logic hooks directly into the game's native Game Rules map. This means **any custom categories or rules added by other mods** will be automatically organized and collapsible out-of-the-box!

### ‚öôÔ∏è Zero Configuration
No config files or messy setup required. It seamlessly integrates into the Vanilla UI and automatically formats everything without needing manual tweaks.

## üìã Quick Start

1. Go to **Singleplayer** -> **Create New World** -> **Game Rules** (or **Options** -> **Game Rules** in-game).
2. See the neatly organized, collapsed categories.
3. Click a category (like "Spawning") to expand it and tweak rules!

---

## üì¶ Install

1. Install **[Fabric API](https://modrinth.com/mod/fabric-api)**.
2. Download `collapsible-game-rules-1.0.0+build.7.jar` and place it in your `mods` folder.
3. Launch the game.
